# ③ Access Networks

> [!abstract] Auf einen Blick
> **Access Networks** sichern den Zugang **vieler** Teilnehmer zum Netz und zu Diensten — über sehr unterschiedliche Technologien. Sie unterscheiden sich auf **Ebene 1** (Daten ↔ Signal) und **Ebene 2** ([[2 Link Layer/2.0 Link Layer (Übersicht)\|Link Layer]]); höhere Schichten sehen einen zuverlässigen Frame-Strom.

## Notes in diesem Kapitel
1. [[3 Access Networks/Access Networks Grundlagen\|Access Networks Grundlagen]] — Aufgaben, Distribution/Feeder, Ebenen
2. [[3 Access Networks/DSL\|DSL]] — Breitband über die Telefonleitung
3. [[3 Access Networks/Ethernet\|Ethernet]] — das wichtigste LAN-Protokoll (CSMA/CD)
4. [[3 Access Networks/WLAN\|WLAN]] — drahtloses LAN nach 802.11 (CSMA/CA)
5. [[3 Access Networks/Mobilfunk\|Mobilfunk]] — zellulare Netze von 1G bis 5G
6. [[3 Access Networks/IoT & Bluetooth\|IoT & Bluetooth]] — kleine, sparsame Funknetze

## Roter Faden
Auf der theoretischen Basis von [[2 Link Layer/MAC – Medienzugang\|MAC – Medienzugang]] entstehen die konkreten Zugangstechniken: kabelgebunden ([[3 Access Networks/DSL\|DSL]], [[3 Access Networks/Ethernet\|Ethernet]]) und drahtlos ([[3 Access Networks/WLAN\|WLAN]], [[3 Access Networks/Mobilfunk\|Mobilfunk]], [[3 Access Networks/IoT & Bluetooth\|IoT & Bluetooth]]).
